20.f stripInaccessible()
üé§INTERVIEW SECTION
30-Second Interview Answer
Security.stripInaccessible() is a Salesforce Apex method used to enforce Field-Level Security (FLS) and

object-level permissions by removing fields from sObjects that the current user does not have access to.

  • It is the recommended way to ensure secure Apex code.
2-Minute Technical Explanation
Problem:
  • Apex runs in system mode.
  • It can access fields even if user does not have permission.
Risk:
  • Sensitive fields exposed via:
  • Apex REST
  • LWC controller
  • Batch job
  • Integration
Solution:
Use:
Security.stripInaccessible()
It:
‚úî Removes fields user cannot access
‚úî Enforces FLS dynamically
‚úî Prevents security violations
‚úî Required for AppExchange security review
Real-Time Scenario
  • User cannot see:
  • Salary__c field.
  • But Apex query includes:
  • SELECT Id, Name, Salary__c FROM Account
  • Without stripInaccessible:
  • Field exposed.
  • With stripInaccessible:
  • Salary__c removed automatically.
  • Secure design.

‚Üí No.

‚Üí Before DML or returning data to UI/API.

‚Üí Yes (recommended).
