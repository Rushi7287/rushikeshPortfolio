30-Second Interview Answer

In Salesforce, metadata is the structure or configuration of the application ‚Äî like objects, fields, page

  • layouts, Apex classes, and flows.

Data is the actual record information stored inside those objects ‚Äî like Account records, Opportunity

  • records, etc.

Salesforce is a metadata-driven platform, meaning behavior is controlled by configuration rather than

  • hard-coded logic.
Metadata
  • Metadata defines:
  • Object structure
  • Field definitions
  • Validation rules
  • Triggers
  • Flows
  • Page layouts
  • Security settings
  • Metadata controls how application behaves.
Data
  • Data is:
  • Records inside objects
  • Values stored in fields
  • User-entered information
  • System-generated information

Why Salesforce Is Metadata-Driven

  • No hard-coded schema changes
  • Admin can create fields without coding
  • Upgrades do not break customization
  • Everything deployable via Metadata API

Custom Metadata vs Custom Settings
  • Custom Metadata
  • Deployable
  • Used for configuration
  • Queried like object
  • Better for production config
  • Custom Settings
  • Org-specific configuration
  • Not easily deployable
Governor Limit Impact
  • Metadata retrieval:
  • Does not count as SOQL query in same way
  • Cached by platform
  • Data queries:
  • Count towards SOQL limits
  • Subject to selectivity rules
Performance Considerations
  • Avoid querying unnecessary metadata repeatedly
  • Use Schema methods efficiently
  • Cache configuration values
Changing Field Type
  • May cause data loss
Deleting Field
  • Data permanently deleted after 15 days
Changing Metadata in Production
  • Can break integrations
Debugging Issues
  • If behavior not working:
  • 1Check validation rule metadata
  • 2Check trigger metadata
  • 3Check flow metadata
  • 4Check field-level security metadata
  • 5Then check record data
Best Practices
  • Use Custom Metadata for configurable logic
  • Avoid hard-coding values
  • Use describe methods for dynamic handling
  • Keep metadata clean and documented
