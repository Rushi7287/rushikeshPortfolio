30-Second Interview Answer

Screen Flow is a type of Salesforce Flow that provides a guided, interactive user interface with screens,

inputs, and decisions. It is used to collect user input, perform logic, and create or update records

  • through a step-by-step wizard experience.
2-Minute Technical Explanation
  • Screen Flow:
  • Runs in user context
  • Requires user interaction
  • Contains Screen elements
  • Can perform DML
  • Can call subflows
  • Can call Apex
Can perform callouts (via action)
