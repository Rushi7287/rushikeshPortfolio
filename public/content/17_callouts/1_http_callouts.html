30-Second Interview Answer

An HTTP Callout in Salesforce is a server-side request made from Apex to an external system using HTTP

or HTTPS. It is used to integrate Salesforce with external APIs for sending or retrieving data. Callouts

  • must be made asynchronously when triggered from DML contexts like triggers.
