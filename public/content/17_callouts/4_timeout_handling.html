30-Second Interview Answer

Timeout handling in Salesforce callouts ensures that external API calls do not hang indefinitely.

Developers can configure timeout values (up to 120 seconds) in Apex and must handle timeout
  • exceptions properly using try-catch and retry strategies to prevent transaction failures.
