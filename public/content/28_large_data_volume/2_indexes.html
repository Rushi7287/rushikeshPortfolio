30-Second Interview Answer

Indexes in Salesforce are database structures that improve query performance by allowing the query

optimizer to locate records efficiently without performing a full table scan. Indexed fields are critical for

  • writing selective queries in large data volume environments.
2-Minute Technical Explanation
In LDV orgs (millions of records):
  • Without index:
Timeout errors
  • With index:
‚úî Fast record retrieval
‚úî Selective query
‚úî Efficient filtering
  • Salesforce provides:
‚úî Standard Indexes (automatic)
‚úî Custom Indexes (via support)
‚úî External ID index
‚úî Unique field index
  • Indexes are foundation of LDV performance strategy.
Real-Time Scenario
  • Org has:
  • 5 million Opportunity records.
Query:
  • SELECT Id FROM Opportunity
  • WHERE Region__c = 'West'
  • If Region__c not indexed ‚Üí
  • Full scan ‚Üí Performance issue.
Solution:
  • Request Custom Index on Region__c ‚Üí
  • Query becomes selective ‚Üí Fast execution.

‚Üí Yes.

‚Üí No (except deterministic special cases).

‚Üí No (must request Salesforce support).
