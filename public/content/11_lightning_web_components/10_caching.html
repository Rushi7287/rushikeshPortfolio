30-Second Interview Answer

Caching in LWC improves performance by storing previously fetched data on the client side to reduce

server calls. It is primarily achieved using @wire with @AuraEnabled(cacheable=true) Apex methods or
through Lightning Data Service (LDS), which automatically handles caching and synchronization.
