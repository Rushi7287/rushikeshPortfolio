üé§INTERVIEW SECTION
30-Second Interview Answer

@wire is used for reactive, cacheable, automatic data fetching in LWC, while Imperative Apex calls are

used for manual, event-driven, or non-cacheable operations. Wire is best for read-only UI data;
  • Imperative is best for create, update, delete, or controlled execution.
2-Minute Technical Explanation
  • Both approaches call Apex, but differ in execution style.
wire
  • Reactive
  • Automatically executes
Requires @AuraEnabled(cacheable=true)
  • Best for read-only data
  • Auto re-executes when parameter changes
Example:
@wire(getAccounts)
accounts;
Imperative Call
  • Manual execution
  • Used inside function
  • Works with cacheable or non-cacheable
  • Required for DML operations
Example:
getAccounts()
.then(result => {
this.accounts = result;
})
.catch(error => {
console.error(error);
});

  • Scenario 3 ‚Äì Search on user input

  • Scenario 4 ‚Äì Auto-refresh when recordId changes
‚Üí @wire
