16.a REST API

üé§INTERVIEW SECTION
30-Second Interview Answer
Salesforce REST API is a lightweight, web-based API that uses HTTP methods (GET, POST, PATCH, DELETE)

to interact with Salesforce data in JSON format. It is widely used for web and mobile integrations due to

  • its simplicity and stateless architecture.
2-Minute Technical Explanation
REST (Representational State Transfer):
  • Uses HTTP protocol
  • Uses JSON format
  • Stateless communication
  • Easy to integrate
  • Common HTTP Methods:
  • Method Purpose
  • GET Retrieve data
  • POST Create record
  • PATCH Update record
  • DELETE Delete record
  • Base endpoint:
  • /services/data/vXX.X/
Example:
  • GET /services/data/v59.0/sobjects/Account/
  • Returns JSON.
Real-Time Scenario
  • External mobile app needs:
  • Fetch Account details
  • Create Opportunity
  • Update Contact
  • Use Salesforce REST API.
  • Lightweight & scalable.
