30-Second Interview Answer

Apex REST is a custom REST API built using Apex classes in Salesforce. It allows developers to expose

custom business logic as REST endpoints when standard Salesforce REST APIs are not sufficient.

2-Minute Technical Explanation
  • Standard REST API:
‚úî Basic CRUD
‚úî Standard object access
  • But when we need:
  • Complex business logic
  • Multi-object processing
  • Custom validation
  • Aggregated response
  • Controlled error handling
  • We use Apex REST.
  • Defined using:
@RestResource(urlMapping='/myapi/*')
global with sharing class MyAPI {
Supports:
  • GET
  • POST
  • PUT
  • PATCH
  • DELETE
  • Returns JSON by default.

  • Standard REST API cannot handle full logic.
