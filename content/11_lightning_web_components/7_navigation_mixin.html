üé§INTERVIEW SECTION
30-Second Interview Answer

NavigationMixin is a utility in LWC that allows navigation to different pages in Salesforce such as record

pages, list views, object pages, web pages, or custom components. It enables programmatic navigation

  • using a standardized PageReference object.
2-Minute Technical Explanation
  • To use NavigationMixin:
import { NavigationMixin } from 'lightning/navigation';
  • Extend component:
export default class MyComponent extends NavigationMixin(LightningElement)
  • Navigation works using a PageReference object:
this[NavigationMixin.Navigate]({
type: 'standard__recordPage',
attributes: {
recordId: recordId,
objectApiName: 'Account',
actionName: 'view'
}
});
  • Used for:
‚úî Redirect after save
‚úî Navigate to related record
‚úî Open new record page
‚úî Navigate to list view
‚úî Open external URL

üî¨DEEP TECHNICAL BREAKDOWN
Step 1: Import NavigationMixin
import { NavigationMixin } from 'lightning/navigation';
Step 2: Extend Component
export default class MyComponent extends NavigationMixin(LightningElement)
  • Without this ‚Üí navigation fails.
Navigate to Record Page
this[NavigationMixin.Navigate]({
type: 'standard__recordPage',
attributes: {
recordId: this.recordId,
objectApiName: 'Account',
actionName: 'view'
}
});
  • actionName options:
  • view
  • edit
  • clone
Navigate to New Record Page
this[NavigationMixin.Navigate]({
type: 'standard__objectPage',
attributes: {
objectApiName: 'Account',
actionName: 'new'
}
});
Navigate to List View
this[NavigationMixin.Navigate]({
type: 'standard__objectPage',
attributes: {
objectApiName: 'Account',
actionName: 'list'
},
state: {
filterName: 'Recent'
}
});
Navigate to Web Page
this[NavigationMixin.Navigate]({
type: 'standard__webPage',
attributes: {
url: 'https://www.google.com'
}
});
Passing Custom State Parameters
state: {
c__customParam: 'value'
}
Important:
  • Custom parameters must start with c__.
  • Access in target component:
import { CurrentPageReference } from 'lightning/navigation';
Generate URL Without Navigating
this[NavigationMixin.GenerateUrl](pageReference)
.then(url => {
window.open(url);
});

Navigation Types Summary
  • Type Use Case
  • standard__recordPage Record view/edit
  • standard__objectPage Object home/list/new
  • standard__webPage External URL
  • standard__component Navigate to LWC/Aura component
  • standard__navItemPage Custom tab
