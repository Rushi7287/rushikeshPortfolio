30-Second Interview Answer

Autolaunched Flow is a type of Flow that runs automatically without user interaction and does not

contain screens. It can be triggered by other flows, Apex, Process Builder (legacy), or invoked as a
  • subflow. It is used for background automation and reusable logic.
2-Minute Technical Explanation
  • Autolaunched Flow:
  • Has no Screen elements
  • Runs in background
  • Can perform DML
  • Can call Apex
  • Can be called from:
  • Record-Triggered Flow
Apex (@InvocableMethod)
  • Subflow element
  • REST API
  • Used for:
‚úî Reusable automation logic
